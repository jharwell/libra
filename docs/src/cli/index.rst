================
CLI (``clibra``)
================

``clibra`` is a preset-aware CLI that wraps ``cmake``, ``cmake --build``,
and ``ctest`` with sensible defaults. It never introduces state that plain
CMake cannot read — you can drop it at any point and drive the build
directly.
