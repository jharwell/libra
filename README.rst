=======================================
Luigi Build Reusable Automation (LIBRA)
=======================================

.. |docs| image:: https://git.sift.net/jharwell/libra/badges/devel/pipeline.svg?job=pages&key_text=docs&key_width=50
          :target: https://jharwell.pages.sift.net/libra

|docs|

This is a repository containing 100% reusable cmake scaffolding that can be used
for nested/flat C/C++ projects (even mixing the two), and provides reusable
build "plumbing" that can be transferred without modification between projects.


Project documentation is `here <https://jharwell.pages.sift.net/libra>`_.
